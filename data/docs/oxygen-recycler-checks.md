# Oxygen Recycler Checks

## Purpose

This document defines recurring checks for the oxygen recycler and related life-support diagnostics.

## Daily Checks

- Perform oxygen recycler diagnostics at the beginning of the operations window.
- Inspect pressure seals before any non-critical expansion task begins.
- Verify that backup filtration status is green before reducing crew oversight.

## Priority Rules

- Oxygen recycler diagnostics are always high priority.
- Pressure seal inspection should be scheduled before greenhouse or construction tasks when crew time is limited.
- If life-support diagnostics are delayed, deprioritize non-essential communications planning.

## Warning Conditions

- If oxygen purity readings are unavailable, mark the plan as uncertain.
- If a pressure seal inspection has not been logged, warn before scheduling exterior or construction work.
- If backup filtration is degraded, recommend follow-up monitoring within the same shift.

## Escalation

- Missing oxygen recycler diagnostics require immediate manual review.
- A failed seal check blocks exterior expansion tasks until cleared.
